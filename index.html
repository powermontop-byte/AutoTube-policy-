Polityka Prywatności

Niniejsza Polityka Prywatności opisuje zasady dotyczące korzystania z aplikacji AutoTube, udostępnianej przez PowermontApps.

Gromadzenie danych

AutoTube nie gromadzi, nie przechowuje ani nie udostępnia żadnych danych osobowych użytkowników.

Aplikacja nie wymaga zakładania konta użytkownika i nie wykorzystuje formularzy rejestracyjnych. Nie są zbierane dane takie jak imię, nazwisko, adres e-mail, numer telefonu ani lokalizacja użytkownika.

AutoTube korzysta wyłącznie z podstawowych uprawnień systemowych wymaganych do poprawnego działania funkcji aplikacji, takich jak:

  • Dostęp do internetu — wymagany do działania funkcji online.
  • Dostęp do pamięci urządzenia — opcjonalnie wykorzystywany do przechowywania ustawień aplikacji lub plików tymczasowych.
  • Bluetooth / Android Auto — używane wyłącznie do funkcji związanych z obsługą samochodową.

Żadne dane użytkownika nie są sprzedawane ani przekazywane podmiotom trzecim.

Usługi zewnętrzne

Aplikacja może korzystać z usług stron trzecich, takich jak usługi Google, które mogą zbierać anonimowe dane techniczne lub diagnostyczne.

Bezpieczeństwo

Bezpieczeństwo użytkowników jest dla PowermontApps najwyższym priorytetem.

AutoTube został zaprojektowany w sposób ograniczający dostęp do danych urządzenia wyłącznie do minimum wymaganego do poprawnego działania aplikacji.

Aplikacja nie monitoruje aktywności użytkownika, nie analizuje prywatnych danych i nie wykonuje ukrytych operacji w tle związanych z przetwarzaniem danych osobowych.

Dokładamy wszelkich starań, aby aplikacja była bezpieczna, stabilna oraz w pełni zgodna z wymaganiami bezpieczeństwa systemu Android.

Privacy Policy

This Privacy Policy describes the rules regarding the use of the AutoTube application provided by PowermontApps.

Data Collection

AutoTube does not collect, store, or share any personal user data.

The application does not require user registration and does not use account creation forms. Information such as name, email address, phone number, or user location is not collected.

AutoTube may only use basic system permissions necessary for proper app functionality, such as:

  • Internet access — required for online features.
  • Device storage access — optionally used for saving app settings or temporary files.
  • Bluetooth / Android Auto — used only for vehicle compatibility features.

No user data is sold or shared with third parties.

Third-Party Services

The application may use third-party services such as Google services, which may collect anonymous technical or diagnostic data.

Security

User security is a top priority for PowermontApps.

AutoTube has been designed to limit access to device data strictly to the minimum required for proper application functionality.

The application does not monitor user activity, analyze private information, or perform hidden background operations related to personal data processing.

We make every effort to ensure the application remains secure, stable, and compliant with Android security standards.
